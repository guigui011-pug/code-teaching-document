# code-teaching-document

> 一个让 AI 助手每次写代码、改代码时，都同步维护一套手把手教学文档的 Skill。

---

## 这个 Skill 是做什么的？

当你的代码能力较弱，看不懂 AI 写的代码时，这个 Skill 会强制要求 AI（Claude / ChatGPT）在每次写代码或修改代码后，同步更新一套结构清晰的教学文档——写得像教材，而不是技术手册，让你能自己从头到尾串一遍逻辑。

---

## 文档体系

安装后，AI 会在项目的 `docs/` 文件夹下自动维护 6 个文档：

| 文档 | 内容 |
|------|------|
| `代码结构说明.md` | 每个文件/模块是干什么的，为什么存在 |
| `模块调用关系.md` | 模块之间怎么互相调用，数据从哪来到哪去 |
| `实验流程说明.md` | 手把手教你怎么从头运行整个项目 |
| `修改记录.md` | 每次改了什么、为什么改、改前改后对比 |
| `问题与解决.md` | 遇到过的报错和解决方案 |
| `论文写作素材.md` | 可直接用进论文的方法描述和实验细节 |

文档超过 800 行时自动分卷（如 `代码结构说明2.md`），避免 AI 上下文截断。

---

## 文件说明

```
├── code-teaching-document.skill   ← 安装到 Claude 使用
└── 代码教学文档规范（通用版）.md   ← 粘贴给 ChatGPT 使用
```

---

## 安装方法

**Claude 用户：**
1. 下载 `code-teaching-document.skill`
2. 打开 Claude → Settings → Skills → 上传文件

**ChatGPT 用户：**
1. 打开 `代码教学文档规范（通用版）.md`
2. 复制全部内容，粘贴到对话开头或 Custom Instructions

---

## 触发方式

安装后，对 AI 说以下任意一句话即可触发：

- "帮我把代码结构写到文档里"
- "我看不懂这段代码，帮我解释一下"
- "更新一下教学文档"
- "教会我，我要能自己串一遍"
